# Embrace Sideboard Product Page

This is a sample product page for the "Embrace Sideboard" on an e-commerce website.

## Overview

The product page includes details such as product image, thumbnails, vendor information, pricing, color and size selectors, quantity selector, and an "Add to Cart" button.

## Getting Started

### Prerequisites

- Web browser (Google Chrome, Firefox, Safari, etc.)


Open the index.html file in your preferred web browser.
Features
Responsive design for mobile, tablet, and desktop screens.
Color and size selection for the product.
Quantity selector with increment and decrement buttons.
"Add to Cart" functionality with an informative message.
Usage
Open the index.html file in a web browser.
Explore the product details, select color and size, adjust quantity.
Click the "Add to Cart" button to see the product added to the cart message.
Responsive Design
The webpage is designed to be responsive across various devices:

Mobile Screen: Up to 767px
Tablet Screen: 768px to 1024px
Desktop Screen: Above 1025px
